C Games

J'ai combiné deux de mes programmes en C en une seule application : un jeu de mémoire et un jeu du pendu. L'utilisateur peut choisir le jeu auquel il veut jouer depuis le menu principal. Les deux jeux ont été développés pour pratiquer mes compétences en programmation C et pour créer des jeux amusants et engageants.

Caractéristiques :

  • Nombre de cartes et de valeurs de cartes personnalisables dans le jeu de mémoire
  • Suivi des scores élevés dans les deux jeux
  • Interface simple et intuitive

Ce que j'ai appris

  • L'utilisation de différentes bibliothèques
  • L'amélioration des compétences de résolution de problèmes
  • Les techniques de débogage
