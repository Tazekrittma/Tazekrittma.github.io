Application météo

Une application météo qui affiche les informations météorologiques actuelles d'une ville, alimentée par WeatherAPI.com

Description:

Cette application météo affiche les informations météorologiques actuelles d'une ville à l'aide de l'API WeatherAPI.com. Les utilisateurs peuvent entrer le nom de la ville et du pays pour obtenir des informations telles que la température, la date, les conditions météorologiques, la direction du vent, l'indice UV, l'humidité et la pression. L'application affiche également une image représentant les conditions météorologiques actuelles. Les informations météorologiques sont affichées de manière propre et organisée, et l'application est adaptée aux différentes tailles d'appareils. L'application est construite à l'aide d'HTML, CSS et JavaScript.

Ce que j'ai appris:

  • Comment utiliser une API pour récupérer des données
  • Comment utiliser JavaScript pour manipuler le DOM et mettre à jour l'interface utilisateur en fonction de la saisie de l'utilisateur
  • Comment créer une mise en page responsive en utilisant CSS
