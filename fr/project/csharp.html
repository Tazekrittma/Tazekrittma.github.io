Programme de gestion des employés en C#

Ce projet est un programme de gestion des employés qui vous permet de gérer une liste d'employés en ajoutant, vérifiant les doublons, vérifiant si un employé existe et en imprimant un organigramme. Le programme a été écrit en C# 7.3 et testé à l'aide de NUnit.

Description:

Le programme de gestion des employés est une application basée sur une console qui a été construite en utilisant C# 7.3 et Visual Studio 2022. Il fournit un moyen simple et efficace de gérer une liste d'employés, vous permettant d'ajouter de nouveaux employés, de vérifier les doublons, de vérifier si un employé existe et d'imprimer un organigramme. Le programme a été développé en utilisant les principes de la programmation orientée objet, avec une classe Employee représentant chaque employé dans la liste. Le programme inclut une gestion d'erreurs et une validation d'entrée pour garantir que seules les données valides sont entrées dans le système. Le programme comprend également une suite de tests complète à l'aide du framework NUnit pour garantir que le programme fonctionne correctement et est exempt d'erreurs. Dans l'ensemble, le programme de gestion des employés est un outil utile pour toute organisation qui a besoin de gérer une liste d'employés. Il est simple à utiliser et peut faire gagner du temps et des efforts dans la gestion des données des employés.

Ce que j'ai appris:

  • J'ai développé un programme de gestion des employés en C# basé sur la console pour gérer efficacement une liste d'employés.
  • J'ai utilisé des principes de programmation orientée objet et des techniques de gestion d'erreurs pour garantir que le programme fonctionne correctement.
  • J'ai implémenté une validation d'entrée pour garantir que seules des données valides sont entrées dans le système.
  • J'ai utilisé le framework de test NUnit pour tester rigoureusement le programme et garantir qu'il est exempt d'erreurs.
