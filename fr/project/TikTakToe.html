Jeu de Morpion

Un simple jeu de Morpion construit en HTML, CSS et JavaScript.

Description :

Ceci est un simple jeu de Morpion en ligne construit en HTML, CSS et JavaScript. Le jeu permet à deux joueurs de prendre tour à tour en marquant des X et des O sur une grille 3x3. Le premier joueur qui réussit à placer trois symboles dans une rangée remporte la partie. Si toutes les neuf cases sont remplies sans gagnant, le jeu est déclaré nul. Le jeu a un bouton "Redémarrer" qui permet aux joueurs de réinitialiser la partie et de recommencer.

Ce que j'ai appris :

  • Comment utiliser les écouteurs d'événements en JavaScript pour répondre aux clics de l'utilisateur
  • Comment utiliser des tableaux en JavaScript pour stocker et manipuler des données de jeu
  • Comment utiliser des instructions conditionnelles en JavaScript pour implémenter la logique de jeu
  • Comment manipuler le DOM en JavaScript pour mettre à jour l'interface du jeu.
