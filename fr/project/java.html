Application Calculatrice Java

Une application calculatrice développée en Java, JavaFX en utilisant IntelliJ et SceneBuilder.

Description:

Cette application de calculatrice a été développée en utilisant Java, JavaFX et l'IDE IntelliJ, avec SceneBuilder utilisé pour concevoir l'interface utilisateur (GUI). L'application permet aux utilisateurs d'effectuer des opérations arithmétiques de base telles que l'addition, la soustraction, la multiplication et la division, ainsi que des calculs décimaux. Le code de l'application se compose de deux parties: le code Java, qui configure l'interface utilisateur et gère les entrées utilisateur, et le code JavaFX, qui définit les composants et la mise en page de l'interface utilisateur.

Ce que j'ai appris:

  • Développer des applications GUI en utilisant JavaFX
  • Utiliser SceneBuilder pour concevoir des mises en page GUI
  • Gérer l'entrée utilisateur dans les applications JavaFX
  • Opérations arithmétiques de base en Java
