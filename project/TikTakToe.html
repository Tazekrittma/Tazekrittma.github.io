Tic-Tac-Toe Game

This project is a Tic-Tac-Toe game implemented in JavaScript. The goal is to get three of your symbols in a row (horizontally, vertically, or diagonally) before the other player does. I developed this game as a way to practice my JavaScript programming skills and to create a fun and engaging game.

Features:

  • Two-player mode
  • Modern design
  • Simple, intuitive interface

What I learned:

  • Implementing game logic in JavaScript
  • Manipulating the DOM to update the game board
  • Using event listeners to detect user input
