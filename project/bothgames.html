C Games

I have combined two of my C programs into one application: a memory game and a hangman game. The user can select which game they want to play from the main menu. Both games were developed as a way to practice my C programming skills and to create fun and engaging games.

Features:

  • Customizable number of cards and card values in the memory game
  • High score tracking in both games
  • Simple, intuitive interface

What I Learned

  • Using different libraries
  • Improving problem solving skills
  • Debugging techniques
